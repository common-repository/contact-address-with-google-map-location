== Description ==

This plugin is a widget that will show contact address or any text as html any contact address and google map using exact location name,city,street address etc.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rbrother-google-map` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
